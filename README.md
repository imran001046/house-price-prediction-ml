# House Price Prediction using Machine Learning

This project is an end-to-end machine learning pipeline to predict house prices using the Ames Housing dataset.

## Project Structure

```
house_price_prediction/
│
├── data/
│ ├── raw/ # Original dataset (not tracked by git)
│ └── processed/ # Cleaned dataset (not tracked by git)
│
├── notebooks/
│ ├── 01_data_exploration.ipynb
│ ├── 02_data_cleaning.ipynb
│
├── .gitignore
├── README.md

```

## Steps Covered
- Data Exploration
- Data Cleaning
- Feature Engineering
- Model Building
- Evaluation
- Git Version Control
